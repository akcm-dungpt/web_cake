トップ > 認知症のケア > 施設の種類 > 4. 介護施設紹介サービス




介護施設を利用しようと思っても、数多くの施設の中から本当に安心できる施設を選ぶのは、なかなか難しいのではないでしょうか。生活に関わることですから、悔いのない選択をしたいものです。
そこで認知症ねっとでは、介護施設をお探しの皆様のために、介護施設紹介サービスを開始致しました。専門の相談員が豊富な経験を活かし、親身にお応えします。

また、認知症ねっと「福祉施設を利用する」では、施設の種類や施設選びのポイントなどをご紹介していますので、あわせてご利用ください。
