嚥下機能が低下した高齢者のためのトロミ補助食品

トロミーナI
トロミーナは無味無臭ですので、食品本来の味・風味を損なわず、おいしく召し上がれます。 少量で十分な粘度が得られ、時間が経っても安定し続けます。冷温両用です。素早くきれいに溶解、ダマになりません。のど切れ、のどこしが良くベタつきません。唾液等による離水等の経時変化がないので、食事介助が大変楽になります。

トロミーナII（加熱用）
嚥下困難がさらに進むとゼリー食になります。 加熱調理時に使用します。その後35℃以下になりますとゼリー状になります。少量でしっかりゼリーにします。ゼラチンのゼリーは安定するまで時間がかかりますが、トロミーナは約30～50分でゼリーに出来ます。三日間程の保存が可能で、経済的です。
