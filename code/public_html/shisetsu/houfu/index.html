はぴね防府

http://www.happine-carelink.net/houfu/
梶原たみえ
はぴね防府は　どんなところ？

防府市は山口県のほぼ中央に位置し、瀬戸内海の美しい海岸線に面しています。

夏は海上からのおだやかな風が吹き、冬はその風が中国山地でさえぎられる瀬戸内海特有の気候に恵まれています。

西の奈良ともいわれているように、昔も今も大切にしてきた歴 史と文化のある町です。

また、瀬戸内海や地域を流れる川でとれる魚介類や、中国山地ではぐくまれた山の幸が豊富です。のどかな田園風景も広がってふるさとを感じさせてくれると ころです。このような、海や山の豊かな自然が防府の笑顔となっています。




BPSD「物取られ妄想」の関わりについて

「物取られ妄想」のため、興奮時に出現してしまう　その対応にスタッフが関わる時間を十分 持って対応させて頂いています。しかしどうしても1対１で対応できる時間が持てない際は、ど うしたらいいか、どう向き合ったらいいかを思案していたので　施設内でその対応を検討しま した。

Ｋ様について　今までの生活は　どのように過ごされてきたのであろうか

山口県の瀬戸内海沿いに生まれ育つ。
独身時代は　家事手伝いをしながら、生け花を習い資格を取られました。
結婚後、はぴね防府がある市内に転居される。当時の生活は　生け花をしたり、花や野菜の 栽培を楽しんでおられました。
また、食べる事が大好きなご主人のためと、お客のために美 味しい食材を求めて　毎日早朝から市場を回っていらっしゃいました。夫婦の二人暮らしで あったが　先にご主人が他界。

入居の経緯

90代、女性、健康状態は老人性痴呆・リウマチ・要介護2 ご主人が他界後　独居生活をされていましたが、熱中症にて入院される。
体調は改善したが 年齢のこともあり、すぐに在宅での生活が難しいとの事で老人保健施設に入所となりました。
しかし期限が近づき退居となるが認知症もあり在宅での生活は難しいと、ご家族が判断され 平成17年10月にはぴねへの入居となりました。

入居後のご様子

在宅の頃より物取られ妄想がありご近所に迷惑をかけた事があったそうです。
ご入居されてからも見慣れない生活空間であったためか帰宅願望があり、預金を誰かに全 部取られたなど、ご不安な気持ちが物摂られ妄想となり、他の方にもその話をされる事が多 く、時々困ったという話もありました。

日が経つにつれ、次第に落ち着いてこられましたが現在でも時折、ご不安になられるのか妄 想が見られるる場合もありますが、ご高齢でいらっしゃるのに視力、聴力も良く、周囲の人や スタッフからの話掛けには冗談を交えた会話で返されユーモアのセンスのある方でいらっ しゃいます。

ご家族の思い

甥さんが　本人のQOLの向上には強い要望(買い物、外出)をお持ちでいらっしゃいます。

物取られ妄想の状態時

お一人で、まるで誰かと会話をしているかのように独り言を話されます。
また、近くに他のご 入居者がおられると、銀行に預金の残額を確かめに行く話しをされます。 すこし興奮状態になると、慌ててスタッフルームに来られ　スタッフにご自身の思いを告げられます。

物取られ妄想の際の関わりについて

K様の話をお聞きした上で、気持ちが他に向くような話しかけやお誘いをしますが、一度思い 込まれたらなかなか冷静になられない事が多いため、まず、ご本人様の意志を尊重し、スタッフが一緒に出掛けます。

外に出られ銀行または警察所に行こうとされるので同行しますが歩を進めるうちに、次第に ご本人様に疲労が見られ、ご自身で断念されてから　はぴねに戻られます。
スタッフは　はぴね（自宅）に戻られたら　気持ちよくお迎えし、お疲れを　ゆっくり癒して頂き ます。

このように、K様の物取られ妄想の不安な気持ちに十分に沿えるよう、スタッフが同行 し、一緒に行動をとるようにしています。

難しい点

妄想による興奮時に、はぴねが集団生活の場である事から、すべての時間に　スタッフ が一対一で、ゆっくり向き合える時間がない場合もあります。
そのような時は、どうしていいか、どうやりくりしたらいいか　検討してきました。

たどり着いた先

ご本人様は　誰よりもご家族様とお会いする時が　一番輝いた目をしていらっしゃいます。
今までの家族の関係や　今までのＫ様の生活を大事にする事が　K様の安心できる人や場と なり、また安心できるリズムを作っていると考えています。 いろいろ検討した結果、時々ご家族様に来訪頂き　妄想の際も一緒に対応してもらう事にしました。

一つの施設の力より　　ご家族様の力を頂き、また地域の力を頂きながら、この　はぴね防 府で生活を送っていただく事に　たどり着きました。 はぴねが、ご利用者様を中心に　ご家族に　地域に安心できる場となるように　環境を整えて いく事が大事だと気がつきました。

現在のご様子

排泄 たまに失禁があるが紙パンツの使用はご本人が望まれないため、普通の下着を着用。居室に蓋付きのバケツを置き汚染下着を入れてもらうようにしています。
ご本人が水洗いして置いておられるので、スタッフが洗濯をしています。
歩行 時々、休憩をとりながらゆっくり歩行。但し、妄想により興奮状態の時は小走り。 その時は、注意して見守ります。
入浴 3回/週ヘルパーの介助により清潔を保ち　気持ちよく過ごされています。
食事 なんでも召し上がり完食。おやつも喜んで召し上がります。
レク　 最近は、手芸には関心を示されなくなったが、レクには参加をされ、特に歌のレクは 喜ばれ他のご入居者より高い声で歌われ周囲の人も驚くほどです。
行事 演芸がお好きで芝居や歌謡ショー、舞踊は、とても喜ばれ疲れも見られないほど 熱中して観賞されます。
外食や花見、観劇など、はぴねの外出行事にも積極的に参加し楽しまれています。
