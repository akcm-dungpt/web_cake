はじめに

便秘気味のB様は緩下剤の服用によって排便コントロールを行っていますが、他の内服薬の量も多く内服を拒否さ れ、いつも嫌な思いをされていらっしゃる。そこで私たちは、B様の内服時の負担を少しでも軽減するために緩下 剤を内服せず自然に排便できる方法はないかと考え、乳酸菌飲料を飲用し腸内環境を整えて自然排便ができる のではとの思いから今回の取り組みに臨みました。　

まずはどの様な乳酸菌飲料が腸内を整えるのか、量は、味 は・・とメーカーにも相談し、飲用する種類を決定しました。緩下剤と併用しながら排便状態を観察し、徐々に緩下 剤の量を減らして行き、最終的には緩下剤内服完全中止ができました。
更にその後は、正常な腸内環境を維持で きる乳酸菌飲料の適正量についても観察を続けてみました。


事例

B様、90歳代　要介護５　認知症高齢者生活自立度Ⅳ　約6年前からアルツハイマー型認知症となられる。 高血圧症、心不全、胃潰瘍 女学校を卒業後、結婚。子供を儲けるが夫が戦死。商売を引き継ぎ営む。6年前にアルツハイマー発症後独居が 困難となり、他施設を経てグル－プホームへの入居となりました。

方法

（1）取り組み開始前の（2007年10月）緩下剤の量と排便状況の確認を行いました。
（2）飲用する乳酸菌飲料をメーカーに相談し選定しました。
（3）緩下剤減量に向けての話し合いを主治医、看護師、スタッフ、ご家族様と情報共有を行い、 薬剤調整と乳酸飲料を並行して始める事を説明、確認、了承を頂きました。
（4）緩下剤不要となった為、排便状態を見ながら 乳酸菌飲料の摂取量を徐々に減らしていきました。


経過

2007年10月より緩下剤と併用しながら乳酸菌飲料を開始しました。　排便状態を見ながら緩下剤の量を徐々に減量。　 緩下剤減量後も自然排便の回数が増え、5月8日より完全に緩下剤の内服中止。 その後も7月まではビフィーネS2本とヤクルト400　2本を午前と午後に分けて飲用を続け順調に自然排便がみられました。

スタッフの共有状況

当初は食事や水分の摂取状況が芳しくなく拒否も多かった。そのため、乳酸菌飲料を飲む時間や温度、ビフィーネＳ・ヤクルトの混合の割合い等、各スタッフが現状分析と工夫をさらに行うための情報共有が重要となり、左記の様式で最適な方法を考え全員で方法の統一化を図りました。

結果

排泄ハイセツに関して
当初は緩下剤と共に浣腸・摘便等を行っていましたが緩下剤に頼らず自然排便が行えるようになっていき、それと共に便失禁もほぼ無くなりました。


食事に関して
取り組み前の食事量は平均して約５割摂取と食欲も無く補助食品を利用し栄養を補っていましたが、排泄がスムーズになってからは食欲が増し、主食１０割・副食９割摂取となりました。食に対するモチベーションも上がってきて、丁度一年後には食形態も刻み食から普通食に戻りました。


その他
39.5kgだった体重が取り組み開始から丁度1年間で46kgまで増加しました。(BMI 19％→22％） 水分摂取量も1日1000ccに届かない日が多くみられていたが、現在では殆ど1200cc前後摂取して下さり、風邪な どの病気に罹患しなくなり、肌もみずみずしく若返り、表情に活気が出て反応が良くなってこられました。


ベストな腸内環境について
ビフィーネS・1本削減後、腹圧介助回数増・排便所要時間延長・便の硬化等の変化がありました。
ビフィーネS・2本に戻し、ヤクルト400を1本に削減し経過観察行ったが若干の改善は見られたものの以前の様な 自然な排便状態にはならなかった。
結果、B様にとって良好な腸内環境を維持しスムーズな排便状態を保つために現状に於いてはビフィーネS・2本、ヤクルト400・2本がベストと判明しました。


Ｂ様の生活環境について
緩下剤が中止になってから　施設のレクレーションとして故郷に帰郷する計画を実行しました。故郷の風や時間のゆっ たりとした流れの中で　ご自身の家やご近所の方との食事会では　思いのほか島の魚に舌鼓をうっておられまし た。そんな体験から　せっかく食事を十分に取ることが可能になったのなら　是非故郷での食欲に見られるように　 どんな食事がお好きなのか　どんな故郷の味が満足されるのかとの思いが今まで以上に湧いてきました。

スタッフの変化
全てのスタッフが乳酸菌飲料摂取状況や排泄状態を強く意識するようになり、排便時間、性状等を以前より細かく 申し送りするようになり、他の入居者様の排泄にも意識を向けられるようになりました。
緩下剤を全て無くす事に至り、乳酸菌飲料により腸内環境が良くなるという本人様が元来持っておられる『生きる 力』を大切にし最大限に引き出す介護方法を見つけることができ、それにより、食欲が増してこられました。
せっかく食欲を増してこられたことから　是非好物を思う存分頂いて欲しいと感じたときに　認知症であるBさんの好物はなんだろうという疑問がここで生じ、「センター方式」の一部をスタッフ、栄養士、ご家族様に協力を頂き、Bさん の好きなものを生活歴と時代背景を追いながら、改めてアセスメントを行いました。

最後に

B様は要介護５で歩行不可・重度認知症の方ですが、今回の取り組みにより身体機能・認知機能共に重度な高齢 者の方でも腸内環境を整えれば便秘も解消できる事がわかりました。排泄がスムーズに行くと食欲も出てバランス の良い食事をする事が出来るようになり、良好なｉｎ－ｏｕｔの循環を行えるようになります。Ｂ様自身がはぴねでの生 活を楽しむためにも心身共に健康な身体を保てるように、さまざま情報を医師やご家族様、ケアスタッフ、看護師、 栄養士から頂き工夫していく事だと実感しました。それは何よりもはぴね別府のコンセプトである『いつも利用者様 の側に寄り添って！』です。

今回の取り組みから日々のケアの中でどのな関わりを行えばご入居者様が望む生活を送れるようになるのか、喜んでくださるか、ベストの状態で生活できるかを考えた結果だと思います。人間が本来持っている力を引き出すための工夫が 最も介護に必要な精神だと思います。
