はじめに

首都大学東京システムデザイン学部からのロボット・セラピーに関する研究協力として、メンタルコミットロボットの「PARO」を当院重度認知症デイケアにて　ご利用者様と一緒にその効果を含めて取り組んでいます

「ＰＡＲＯ」とは

タテゴトアザラシの赤ちゃんをモデルにしたアザラシ型ロボット。人と共存するロボットで、かわいいや心地よいなど人からの主観的な評価を重視し、人との相互作用によって、人に楽しみや安らぎなどの精神的な働きかけを行うことを目的に作らています。

視覚(光センサー)、聴覚(マイク)、触覚(面触覚センサー、ヒゲ触覚センサー)などがあるので、光の変化を感じたり、言葉を理解したり、撫でられていることを認識したりして、自律的に反応できる。「世界一の癒しロボット」としてギネスブックに認定されました。

方法

認知症の方は不安や混乱の中にいることが多く、当デイケアご利用者様の大半もその問題を抱えています。特にプロ グラム外の時間の過ごし方でプログラムヘの導入やその日のデイケアの過ごし方が変わってくることが多くあるます。 そのため、朝の開始前の待ち時間や昼食後の休憩時間、帰宅準備中などにPAROの導入を試みました。

反応と効果

大半のご利用者様がPAROの表情や動き、鳴き声にうれしそうな反応を示されます。犬や猫などの動物を飼っていた り、子供好きの方の良い反応が特に目立って起こりました。積極的に撫でたり抱っこをしたりする方と側で見ているだ けの方に分かれますが、側で見ているだけの方でもスタッフからの促しがきっかけで、拒否なくPAROに接することが 多くなられました。

ご利用者様の効果

周辺症状《BPSD》興奮や不安、帰宅欲求に対して出現を抑えられ、 安定した精神状態で過ごすことができました。

スタッフ側の効果

スタッフ側からの利点として、ご利用者様とのコミュニケーションを取 るきっかけや関わりの取っ掛かりとしてのし易さが挙げられました。 PAROは危険性がなく、スタッフの介入が無くともPAROを介して のご利用者様同士のコミュニケーションが得られやすく、また、PAROと接しているご利用者様が穏やかになることで、スタッフがPAROに興味をもたれない方への関わりを持つ時間を得られました。

まとめ

PAROに対しての反応や効果は、ご利用者様・スタッフ双方から見ても良好な面が多くみられました。しかしながら、PAROはただ単に渡せば役に立つもの、というわけではなかった。渡すときの声かけ、渡し方、タイミング、雰囲気作りなどを少し心がけることで癒しの効果をもっと発揮していくことができ、良い関係つくりのきっかけになると感じています。

「PARO」の機能

パロは、このタテゴトアザラシの赤ちゃんのように、柔らかで、さわり心地の良い人工毛皮で全身を覆われ、体にもやわ らかさがあります。 タテゴトアザラシの赤ちゃんは、一日の多くを寝て過ごしていますが、パロにも、朝・昼・夜のリズムがあり、眠くなった り、活発に行動したりします。
パロ には、視覚、聴覚、触覚、運動感覚などがあり、ふれあう人や環境の状況を感じま す。光の変化を感じたり、名前を学習したり、挨拶や褒められる言葉などを理 解したり、なでられたり、叩かれたり、 抱っこされたりすることを感じます。

このような人とのふれあいから、パロにも心や感情があるかのように内部の状態が 変化し、反応の仕方が変わった り、鳴き声を出したり、飼い主の好みの行動を学習したりします。パロの鳴き声は、本物のタテゴトアザラシの赤ちゃん の鳴き声 をたくさん使っています パロは、瞬きをしたりすることで顔の表情に変化があり、頭や手足が動くことで、驚いたり、喜んだり、あたかも心や感 情があるように振舞う自律型ロボットです。パロは、優しくなでられたり、抱っこされたりすると、うれしく感じ、叩かれる と怒っ たりします。ひげを触られるのは、恥ずかしがります。

世界の「ＰＡＲＯ」へ
デンマークでは、高齢者居宅介護促進プロジェクト「Be Safe」の取り組みにおいて、12体のパロを使った7カ月の実証 実験が行なわれ、セラピー効果について高い評価を得たことから、在日デンマーク大使館とデンマーク技術研究所 (DTI)は　産業技術総合研究所が開発したアザラシ型メンタルコミットロボット「パロ」を、同国の高齢者向け施設で本格 導入していくことを11月20日(木)、発表した。
