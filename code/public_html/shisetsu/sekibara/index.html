重度認知症デイケアのメリットについて

デイサービスやショートステイは　使っているけれど　重度認知症デイケアは耳慣れないのだけれど、、、

実は　重度認知症デイケアは　介護保険が始まる前から　歴史は古く　利点がとても大きいです。重度認知症デイケアとデイサービスは共に要介護状態の高齢者が、施設に通って、心身の機能維持回復を図る という目的は同じですが、下記のような違いがあります

重度認知症デイケア デイサービス
  1. 医療機関〔病院、診療所など〕に併設されています。

  3. 作業療法士などがリハビリテーションを実施します。

医師、看護師、または作業療法士、精神保健福祉士、介護スタッフなど　多種の専門家による診療とケアが行われます		 デイケアの1.～3.のような条件はなく、
要介護者が 同じ立場の人とコミュニケーションをする機会を増やし、社会的孤立感の解消や、心身のリフレッシュ、老化の防止などのほか、家族の介護軽減など、
幅広い目的で利用していただける施設です。

Aさんが重度認知症デイケアに来られるまで、、

90歳前、女性10年前に夫と死別。
子供が3人いるも遠方にて暮らしているので同居は難しい状況でした。

一人暮らしをされていいらっしゃいましたが、一ヶ月前程から姿は見えないと近隣の方が心配し、民生委員へ相談がありました。
民生委員の方がAさん宅にお伺いをしたところ、一人暮らしのため　食事もままならず、部屋も掃除されない状況で一人部屋の中に一日中籠もっていらっしゃる状況だったため、地域包括支援センターに連絡し、再度相談員がご本人様と会う状況となりました。

その結果、認知症ではないかということで　当クリニック連絡が入り、今後についての話し合いがもたれることに至りました。

そこで　まずは週1回の精神科の往診が入り、Aさんとの関わりを深めていくことで健康を取り戻し、生活のリズムが整えることから始めました。薬の服飲もあり　訪問看護を通じて健康的な生活と身体を取り戻すことを目標としました。

1年半ほどの時間は要しましたが　根気良く訪問させて頂き、次第に　Aさんの状態もよくなってきたことから、Aさんに、外出する機会ももてるように　往診ではく、デイケアに通って頂くことにしました。

何年かぶりの外出ということから、体調の管理や精神面でのサポートが当面は必要となります。重度認知症デイケアは　医師をはじめとしたスタッフが心身面での支援を専門的に関わり、Aさんの状況を把握しながら治療とケアを行えるというところが　大きな特徴といえます。

いきなり　社会生活に溶け込むのではなく、徐々に専門家の判断、支援の基盤があって　AさんのQOLを高めていくことの出来るようになったのが現在の状況です。

最後に

認知症は現在の所　有効な治療法が見つかってはおりません。
中核症状（記憶障害、理解･判断力の低下など）の進行を治す、あるいは止めることは出来ません。

しかし早急に正しい対処が出来れば妄想･興奮などの周辺症状は比較的抑えられ、穏やかに家での生活を送る事が出来ます。在宅生活を続けていく上では、ご本人様もさることながら介護者の体調・精神の安定や時間的余裕も大切なことです。

重度認知症デイケアは　そのような期待にも十分応えることが出来るところです。
