足立区は　どんなところ？

足立区（あだちく）は東京都の特別区のひとつ。
東京都の北部に位置し、隅田川と荒川に挟まれた地区と、面積の大半東京都の北部に位置し、隅田川と荒川に挟まれた地区と、面積の大半を占める荒川以北の地区とに別れている。南東の荒川南岸に位置。

人口　約635,000人
面積　約53km2

「金八先生」の舞台

足立区立第二中学校は、東京都足立区にあった中学校。3年B組金八先生のロケ地（パート1、5～7）として全国的に有名な中学校であったが、著しい少子化・ドーナツ化現象・中学受験の過熱化などの理由により学区内の生徒数が急激に減少し、全校生徒が127人と少なくなったため、2005年3月末に廃校となった。

はじめに

何らかの理由にて通所をすることになった認知症のご利用者様にとって生活パターンの変化、新しい環境での生活は周辺症状を悪化させることがまれにみられます。今回、デイサービス開始当初より周辺症状が強く見られたアルツハイマー型認知症のご利用者様に対し、医療（デイケア）、介護（デイサービス）の2つの視点から利用者を観察し、治療・ケアを行うことによって症状の緩和がみられ、その人らしく安心安楽に在宅生活を送れた例を取り上げ、周辺症状に対する取り組みをご紹介します。

事例紹介

S様　女性80歳　要介護3　独居　遠方に住む娘がまめに様子を見に来ています。
  • 気が強くプライドが高い
  • 若い頃は自営業、現在はアパート経営をしている
  • 趣味カラオケゲートボールだが、認知症が進行し、周りとの交流が少なくなっていった。ホームヘルパーのみを安否確認のために利用されていらした
  • ご家族様の意向…将来的にはグループホームヘの入居を希望していらっしゃいます

デイサービスから　始めて、、

1月 日中の安全確保目的、集団での生活になれるため、週3回のデイサービス利用を考え、関原クリニックデイサービスの利用をはじめられました。

初参加日は、帰宅願望出現するも、スタッフの声かけで何とか一日過ごされ、帰宅されるがその後の参加からは、スタッフがS様の傍から離れると帰宅願望出現し、また、他者との交流に拒否的になり、他ご利用者様と離れた場所で個人テーブルでの対応となりました。
2月 1日中落ちつきなく、椅子に座らなかったり、レクレーションでのゲームに負けてしまうと、そのことに腹を立て、プログラムに参加しませんでした。

帰宅願望出現の際、スタッフの説得する言葉に興奮、非常口を開けようとし、制止する行為に『殺される～』と、窓を開け、身を乗り出し、外へ向かって助けて～！と叫んだり、スタッフに対して暴力、暴言、破壊行為がみられ、『なんで私はここにいるの？』と不安感、『皆の輪に入れない、私を無視している』との被害妄想もみられ始めました。一方、朝から自転車にて徘徊されたり、自力で来所されることもありました。

デイサービスではその都度、長女へS様の日々の様子を報告し、かかりつけ医へS様の状態を相談していただけるように働きかけました。
　
2ヶ月経過して、、、

S様の行動を認知症の周辺症状（BPSD)と捉え、対応しました。


結果

その結果、落ち着いてプログラムへ参加できる機会が増え、他のご利用者様との交流も行えるようになりました。
しかしBPSDは　続いていた。

ご家族様からの希望
  • S様の主治医の変更が短期間で続いており、ご家族様の話をきちんと聴いてもらえない、病院を 変更したい
  • 介護保険の単位をオーバーし、実費が発生していた
  • 通所の利用回数を増やしたい

デイケアにて

9月 デイケア初参加

2ヶ月間のデイサービスでのS様の情報を　クリニック・デイケアの医師と情報共有し周辺症状の悪化と思われる場面も増えたので、服薬調整の検討をご家族様同意の基に始めました。また、服薬を調整するにあたり、各関係機関へも連絡し、調整後のS様の状態の観察を依頼、変化等あれば、デイケアへ連絡いただけるよう伝達しました。

服薬調整とあわせて、デイケア、デイサービスでのS様に対するケアの統一性をはかることとなり、院内でも日々の状態を細かく申し送ることを徹底しました。

デイサービスにて

サービス変更後のデイケア、デイサービスでの周辺症状に対しての取り組みとして、バリデーション療法（受容、共感）を取り入れました。

♪バリデーションとは・・・
「認知症の人をそのまま受け入れること」認知症の問題行動と捉えられていたことも、その人なりの意味があるとしてケア者は受け入れる。「受け入れて、共感する」為のコミュニケーション技術として考案された技法で周辺症状がゼロになるわけではないが相手も心を次第に開き始めます。

S様も継続した通所に慣れてきたのか、表情の硬さも少なくなり、笑顔が見られ、他者との交流が行えるようになり、初めの頃よりみられていた帰宅願望、暴力、暴言等は次第に少なくなってきました。そして、対応する職員もS様の表情や行動等、その時々の表情を汲み取り、統一したケアをデイサービス・デイケアで行えるようなりました。

プログラム内容

音読法

小学生の頃から馴染みの深い音読法には、一般的に3つの効果があるとされます。子供から　大人　高齢者まで　万能です。
  • 声を発することにはストレス発散効果や精神の安定効果があります
  • 集中力が高まり，理解力や記憶力向上などの能力開発の効果も現れてきます
  • 声を出すことで元気になるということです。しっかり音読すれば，まず脳が熱くなってきて次第に体全体も暖まっていきます。
頭と体が活性化してきて，活力が湧き元気になることが実感できます。

これは，「元気だから声が出るのでなく，しっかり声を出すから元気になる」という感覚です。

また，音読で声を出し口を動かすことは，唾液の分泌を促し胃腸にもよいと言われています。






回想法

「回想法」とは、「過去の経験や出来事を、想い出し語る(再体験する)ことをとおして、人生をふり返る機会とし、これまでの自分の人生を肯定的に捉え、これからの生活を充実して暮らすことに活かす」

それから

デイサービス・デイケアと連携を密に図り、情報を共有し合うことで関わりが同じになり混乱を防ぎ、安心して生活することに繋がりました。
  1. 医師の診断及び適切な対応方法、指示を受けたことによりS様の状態をスタッフもより詳しく観察、把握し、ケアについて検討ができました。
  2. デイケア・デイサービスで日常の様子を医師も実際に知ることができ、薬の微調整が行えました。
  3. 日々の様子についてもご家族様・医療・介護のスタッフ・ケアマネージャー・ヘルパー間と月一回のカンファレンスで互いに情報交換しケアにつなげることができた。
    結果、S様の全体像を統一した見解で観察、ケアすることによってその人らしく在宅生活を送ることができています。


まとめ

ご利用者様、患者様の情報を共有し、周辺症状に対して薬物療法だけでなく、まずはご本人様の置かれている環境を観察し、何がその周辺症状を引き起こしているのかを考えていきます。その上で関わっている人たちで出来ることを試行錯誤しながら方法を考えます。それでも周辺症状が治まらない場合にはご家族様、医師とも相談しながら少しずつ服薬調整をしていきます。次第に周辺症状が緩和されご本人様も苦痛がなく望まれていた在宅生活を行えるようになっていきます。そのためには、患者様のより詳しい日常生活での情報が必要となり、ご家族様だけの視点ではなく、関わっている人達すべての情報が必要となります。

この事例では在宅生活を送るにあたり、各関係機関がそれぞれの役割を持ち、連携を取りながら、生活を支えています。

周辺症状が激しいからすぐに服薬するというのではなく、在宅生活を送る上で、どのような対応が効果的か、関係機関はどう関わっていくか、周りの人達はどういった役割があるのかなど検討し、実行していくことが大切であると事例を通して学びました。
