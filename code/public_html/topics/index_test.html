認知症ケア専門士単位認定講座
『認知症の周辺症状（BPSD）への心理的サポート』～尊厳を守るケアとは～＆事例検討会 開催のご案内

この度、株式会社ケア・リンク主催、日本認知症ケア学会後援のもとで、認知症に関するセミナーと事例検討会を開催することとなりました。

現在、認知症ケアに取り組んでおられる医療・介護従事者（医師、看護師などの医療スタッフ、ケアマネジャー、ケアスタッフの方）を対象としたセミナー［受講料無料］です。

なお、認知症ケアに関心のある皆様のご参加も歓迎です。皆様のご来場を心よりお待ち申し上げます。

※認知症ケア専門士をお持ちの方には受講の場合2単位が加算されます。
