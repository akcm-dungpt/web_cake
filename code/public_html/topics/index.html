音楽療法県民フォーラム 「音楽の力をすべての人に」のおしらせ

兵庫県では、平成11年度に音楽療法講座を開設し、県独自資格である兵庫県音楽療法士の養成を開始し、平成13年度に最初の認定を行いました。 その認定も今年度で10回目を迎えます。

そこで、日本音楽療法学会学術大会が神戸で開催されることにあわせて、県民フォーラムを学会と共催し、これまでの県の取り組みをPRするとともに、音楽療法の原点と課題を、みなさんと一緒に考えます。
