レビー小体型認知症
レビー小体を伴う認知症　
（DLB；Dementia with Lewy Bodies）

発祥する年代
老年期に発症し、男性は女性の2倍多く発祥します。

原因
脳幹や大脳皮質にレビー小体という異常な細胞内蓄積されます。

症状
初期
体の動きが緩慢になるパーキンソン病に似た症状で、歩行の障害や体の硬さともなうため、転倒しやすくなります。

色彩を伴う非常に鮮明な人・動物・虫などが昼夜問わず出現します。

幻視の人物が話したりするなどの幻聴もあり、ＲＥＭ睡眠行動障害により　睡眠時に夢にあわせて踊ったり、手足を動かしたり、歩いたりする行動もとってしまうことがあります。

認知障害も変動しやすく、良いときは話が通じるが、悪くなると話も周りのこともわからなくなり、一貫した行動が取れなくなります。

大事なこと
薬の調整を現状に合わせて行うことが大切です。
アルツハイマー型のアリセプトで　幻視が消えない場合は　レビー小体では、手足の振るえなどを招く恐れのある抗精神薬は使用できない。漢方薬の抑肝散が効果があるとされています。
