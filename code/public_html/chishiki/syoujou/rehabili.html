認知症のリハビリテーション

認知症のリハビリテーションは病院で訓練されるものではありません。

2001年に世界保健機構で国際生活機能分類（ＩＣＦ）が採択されました。

ＩＣＦでは心身機能だけでなく、社会に活動し参加していくことを視点に重要視しています。

認知症の疾患原因（脳血管性認知症）に対する　心身機能の障害（左半身麻痺）を了解のもと社会生活をいかに送るか　参加していくことを考えること（ディサービスを利用する、ヘルパーさんと買い物に行くなど）が大切です。

作業療法・回想療法

地域の夏祭りがあるために浴衣を縫うことを通じて、生地の裁断はできなくとも運針は手続き記憶がありできる場合など、できる部分の作業を行うことで残存機能を生かすことができます。　

ご本人の好きな手仕事を行うことで達成感や満足感を得るという心理面の安定をもたらします。

作業を通じて、昔の夏祭りの思い出や家族とのかかわりを参加者と共に回想することにより、孤立感を癒し心の安定をもたらすと主に人生の統合を目指すアプローチとされています。

理学療法
徘徊があり、帰宅願望がある方にたいして、毎日の散歩を日課にすることがあります。　

理学療法の中の運動療法として有酸素運動の効果としては高血圧や糖尿病の予防につながり、精神面でも外出をすることで地域社会に関わり孤立感を持たずに安定が図ることができます。

音楽療法
音楽を聴いて　安らいだり楽しくなったり　その音や歌を聴くことでその当時の思いでも浮かんで充実した時間を持つことができます。

歌うことでおなかから声をだし、楽器を弾くことで体をうごかすなど　心身ともに機能の向上も図れます。

バリデーション
バリデーションとは、アルツハイマー型認知症および類似の認知症の高齢者とコミュニケーションを行うための方法の一つです。

認知症の高齢者に対して、尊敬と共感をもって関わることを基本とし、お年寄りの尊厳を回復し、引きこもりに陥らないように援助するコミュニケーション法です

タクティールケア
タクティールとはラテン語で「触れる＝タッチ」という意味ですが、手足あるいは全身に対し痛点を刺激することなく手で触れ、興奮状態や不安感、痛みなどを緩和します。
